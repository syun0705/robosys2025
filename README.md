## 概要

- このプログラムは、ローンを組むときの返済額をシュミレーションするプログラムです。
- "借りる金額"、"期間"、"金利"を入力することで、"月々の返済額"、"年間返済額"、"総返済額"のそれぞれを出力します。
- 借りる金額は、～万円から入力されます。(20万円の場合、入力する値は"20")

## 必要なソフトウェア

- python  テスト済みバージョン:3.8 ~ 3.11

## テスト環境

- Ubuntu 24.04.3 LTS

## 事前準備

```
git clone https://github.com/syun0705/robosys2025.git
cd robosys2025
```

## 実行方法

```
echo "借りる金額 期間 金利" | ./loan_money
```
"借りる金額"、"期間"、"金利"はそれぞれ必要な値を入力して下さい。

## 実行例

"借りる金額:2000万円"、"期間:30年間"、"金利:2.0%"とする場合

```
echo "2000 30 2.0" | ./loan_money
月々の返済額：73,924円
年間の返済額：887,087円
総返済額：26,612,602円
```
